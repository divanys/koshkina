Проект представляет собой простую модель магазина. В проекте созданы классы для продуктов,
клиентов, кассиров, заказов и магазина.

Класс Product:

Класс Product представляет продукт в магазине и содержит информацию о названии, производителе, цене,
количестве в упаковке/массе нетто, дате изготовления и сроке годности.
Продукты могут быть зарегистрированы в магазине и имеют метод для расчета скидки.

Класс Customer:

Класс Customer представляет клиента магазина и содержит информацию о фамилии, имени, отчестве и возрасте клиента.

Класс Cashier:

Класс Cashier представляет кассира магазина и содержит информацию о фамилии, имени, отчестве и возрасте кассира.

Класс Order:

Класс Order представляет заказ в магазине и содержит информацию о клиенте, продуктах в заказе и примененной скидке.
 Заказы могут быть добавлены в магазин и могут применять скидки к продуктам.

Класс Store:

Класс Store представляет сам магазин и содержит списки продуктов, клиентов, кассиров и заказов.
 Магазин позволяет добавлять продукты, клиентов, кассиров и заказы, а также применять скидки к заказам.
  Также в магазине реализован метод для очистки списка продуктов.

Класс Main:

Класс Main содержит метод main, который создает объекты продуктов, клиентов, кассиров и заказов,
а затем использует методы класса Store для добавления их в магазин и проведения операций с ними.
Проект позволяет продемонстрировать функциональность магазина, включая добавление объектов, применение скидок и обработку заказов.

методы:
addProduct(Product product) - Добавляет продукт в магазин, предотвращая дубликаты на основе уникальных характеристик продукта.

clearProducts() - Очищает список продуктов в магазине.

addCustomer(Customer customer) - Добавляет клиента в магазин, проверяя наличие дубликатов по ФИО клиента.

addCashier(Cashier cashier) - Добавляет кассира в магазин, проверяя наличие дубликатов по ФИО кассира.

addOrder(Order order, int discountPercentage) - Добавляет заказ в магазин, применяя указанную скидку к продуктам в заказе.
Проверяет наличие дубликатов заказов для одного клиента с одними и теми же продуктами.

printInfo() (в классе Product) - Выводит информацию о продукте, включая название, категорию, производителя, цену,
 количество в упаковке/массу нетто, дату изготовления и срок годности.

calculateDiscount(int discountPercentage) (в классе Product) - Рассчитывает цену продукта с учетом скидки, применяя
 процент скидки к цене.

printOrderInfo(int discountPercentage) (в классе Order) - Выводит информацию о заказе, включая клиента, продукты в заказе
и общую стоимость заказа с учетом скидки.

applyDiscount(Product product, int discountPercentage) (в интерфейсе Promotion) - Применяет скидку к указанному
продукту в рамках акционного предложения.

getPromotionDetails() (в интерфейсе Promotion) - Возвращает информацию о текущем акционном предложении.

Метод regProduct(int num) предназначен для регистрации продукта в магазине.
При вызове этого метода, продукт помечается как зарегистрированный в магазине и присваивается количество
в упаковке (или масса нетто) на основе переданного аргумента num.
Если продукт уже зарегистрирован, метод выдаст сообщение о том, что продукт уже имеется в магазине.
